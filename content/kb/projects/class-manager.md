# Class-Manager

위치 기반 QR 출석 서비스

## 소개

강의 출석을 모바일로 간편하고 정확하게 처리하는 목표로 진행한 프로젝트입니다. QR코드 스캐너와 딥링크 활용으로 편리한 사용자 경험을 제공하고자 하였고, 부정 출석 방지 절차로 신뢰성 있는 출석 프로세스를 구현하기 위해 노력하였습니다.

- **형태**: 개인 프로젝트
- **참여 역할**: 기획, 설계, 개발
- **주요 기술**: React Native, Nest.js, MongoDB

## 경험

- 카메라, 위치, 디바이스 정보 등 Native API 활용
- 인증 절차와 역할 기반 제어 구현
- NoSQL 인덱스를 활용하여 수업 정보 조회 쿼리 반환 시간을 10ms 수준으로 확보
- OAuth 기초 이해 및 소셜 로그인 구현
- Refresh Token 전략 적용
